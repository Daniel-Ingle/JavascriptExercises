Tile Memory Game

This game is a tile game where letters are underneath the tiles. A player selects two tiles and, if they match, they stay opened. Otherwise, if they don't match, the tiles are flipped closed again and the player selects two more to select matching tiles. The player wins when all of the tiles have been flipped over. Score is based on how quiclky they were able to match all of the tiles.

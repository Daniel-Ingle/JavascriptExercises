Make a quiz game to answer what the capital is for each state in the U.S. The user should be presented with a state (in the 'stateDisplay') and will need to answer what the capital of that state is. When they hit enter or the 'Guess' button, it will:

  • Show if the answer was correct, or, if wrong, tell the user the right answer.
  • If they guessed correctly, increment the score.
  • Show the next state to try to answer.

The state's and their capitals are saved as an array in the stateCapitals.js file. The app.js file imports this array for you to use there.

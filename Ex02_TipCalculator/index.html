This will help you calculate your tip, based on your total bill and the amount you want to tip, from 0% to 30%. The "Tip" slider allows you to select a value in the range of 0 through 30 as its value. The ".value" property of this input will return that value as a number, 0 to 30. Tip amount should be "Total Bill" times the "Tip" as a percent (the value of the input divided by 100), and the Total with Tip is this plus the original "Total Bill".

Hints:
  • The slider is an input type, with a value based on the values 0 to 30. You can get the value just as if it were a regular "text" or "number" input.
  • The tip isn't the value of the range input field, it is the value / 100, to make it a percentage.
  • We'll need to handle the 'change' events on both the input range field for the tip, plus the Total Bill input. The listeners have been provided, but you'll need to create your own function.
